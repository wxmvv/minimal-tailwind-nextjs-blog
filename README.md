# minimal-tailwind-nextjs-blog

简单的Next.js博客模板。

基于[Tailwind Nextjs Starter Blog](https://github.com/timlrx/tailwind-nextjs-starter-blog)。

## 开发

```bash
yarn install
yarn dev
```

## 部署

```bash
yarn build
```
